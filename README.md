## About:

This repo contains patches for `st`, `dwm`, scripts, and configuration files for various pieces of software.

## Using:

Clone the repo:
```bash
$ git clone git@github.com:Mulling/dots.git
```

Deploy the config files:
```bash
dots/deploy
```
